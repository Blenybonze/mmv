# Memórias Manchadas de Vermelho - Landing Page

Landing page moderna para o lançamento do livro "Memórias Manchadas de Vermelho" de Manoelly Deodoro.

## Como baixar e rodar o projeto

### Pré-requisitos
- Node.js (versão 16 ou superior)
- Git

### Passos para executar

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/Blenybonze/mmv.git
   ```
   Ou baixe o ZIP pelo botão "Code > Download ZIP" e extraia em sua máquina.

2. **Acesse a pasta do projeto:**
   ```bash
   cd mmv
   ```

3. **Instale as dependências:**
   ```bash
   npm install
   ```

4. **Inicie o servidor de desenvolvimento:**
   ```bash
   npm run dev
   ```

5. **Acesse no navegador:**
   - O projeto estará disponível em `http://localhost:5173`

---

## Observações
- Caso não tenha o Node.js instalado, baixe em: https://nodejs.org/
- Para instalar o Git: https://git-scm.com/
- O projeto é totalmente front-end, não requer banco de dados ou configurações adicionais.
- Para dúvidas, consulte o código-fonte ou entre em contato com o autor.
